# 💓 FitPulse — Health Anomaly Detection Dashboard

FitPulse is an interactive health analytics dashboard that detects anomalies in fitness tracker data such as heart rate, steps, and sleeping hours using intelligent rule-based analysis.

---

## 🚀 Features

* 📊 Interactive dashboard built with Streamlit
* ⚠️ Detects anomalies in real-time
* 📈 Visualizations using Plotly
* 📂 Upload CSV/JSON fitness data
* 📅 Filter by date range
* 📥 Export anomaly reports

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* NumPy
* Plotly

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run modules/milestone4_dashboard.py
```

---

## 📂 Input Format

Required columns:

* `timestamp`
* `heart_rate_bpm`
* `steps`
* `sleeping hours`
